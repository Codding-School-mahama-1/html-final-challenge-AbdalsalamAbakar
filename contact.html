Feature Description
Internal Link A hyperlink that points to another section within the same webpage or document.
External Link A hyperlink that points to a different webpage or document, often on a different website.
Anchor Tag An HTML element () used to create hyperlinks, with the "href" attribute specifying the link's destination.
